 ## Dependency
 
 Add this in your root `build.gradle` file (**not** your module `build.gradle` file):
 
 ```gradle
 allprojects {
 	repositories {
         maven { url "https://jitpack.io" }
     }
 }
 ```
 
 Then, add the library to your module `build.gradle`
 ```gradle
 dependencies {
     implementation 'com.github.Sum-sdl:RvAdapter:2.0.1'
 }
 ```
